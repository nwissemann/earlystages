3. website hosting and server log files

When you access this website, the hosting provider automatically collects and stores information in server log files that your browser transmits automatically. These include:

  • IP address of your device,
  • date and time of access,
  • name and URL of the accessed file,
  • referrer URL,
  • browser type and version,
  • operating system.

This data is processed to ensure a stable connection and system security.

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in secure and technically error-free provision of the website).

Storage period: Server log files are automatically deleted after a limited period (typically 7–30 days).

Data recipient: GitHub Pages (GitHub, Inc.) and Cloudflare, Inc., which are used to deliver and secure this website.
